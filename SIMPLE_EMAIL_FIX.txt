# Simple Email Fix for Production

## Quick Fix - Update your .env file with this:

```env
MAIL_MAILER=smtp
MAIL_HOST=smtp.hostinger.com
MAIL_PORT=465
MAIL_USERNAME=hell0@digitalxbrand.com
MAIL_PASSWORD=Avinash!08!08
MAIL_ENCRYPTION=ssl
MAIL_FROM_ADDRESS=hell0@digitalxbrand.com
MAIL_FROM_NAME="FlippDeal By DigitalXBrand"
```

## What changed:
- Port: 587 → 465
- Encryption: tls → ssl

## Steps:
1. Update your production .env file with the above
2. Run: `php artisan config:clear`
3. Test registration

This should fix the SSL connection error with Hostinger.
